# UI-кіт — Аналітичний звіт

Готове до друку редакційне відтворення звіту DIMS (`templates/report.html`).

- **`index.html`** — статичний, самодостатній. Завантажує `styles.css` (для редакційної палітри `--paper`/`--ink` + Source Serif 4 / Inter / JetBrains Mono) та `report.css`.
- **`report.css`** — повна редакційна розкладка: шапка, KPI-сітка, резюме з подвійною лінією, нумеровані рекомендації, картки підозрілих пар, таблиця індикаторів, матриця відстаней Δ, блок методології, футер.

Заповнено зразковими даними, узгодженими з кітом робочого середовища (RT / ТАСС / Sputnik, грейд **SS**). Спроєктовано «друк передусім» — **Друкувати / PDF** відкриває діалог друку браузера; `@media print` згортає хром і робить грейд-бейджі контурними для чорно-білого виводу.
