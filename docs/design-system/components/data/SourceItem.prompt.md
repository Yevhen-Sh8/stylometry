Одне завантажене джерело у списку корпусу (ліва панель). Бейдж типу + назва + моно-мета + кнопка видалення.

```jsx
<SourceItem type="TELEGRAM" label="rt_russian (об'єднано)" meta="t.me · 4 210 ток. · 09:14" onRemove={...} />
<SourceItem type="PDF" label="Звіт ТАСС" meta="312 ток." warn />
```

- `type` забарвлює провідний бейдж: GOOGLE / RSS / TELEGRAM отримують канальні тони; TXT/PDF/URL нейтральні.
- `warn` робить мету бурштиновою — канонічний сигнал «< 500 токенів, низька впевненість».
- Складайте з проміжками 6px; стан ховера — бордюр + підйом `--bg-2` (додайте у своєму контейнері).
