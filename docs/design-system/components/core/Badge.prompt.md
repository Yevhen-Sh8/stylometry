Компактний моно-чіп для лічильників, статусів і коротких тегів. Для грейдів загрози вживайте `GradeBadge`.

```jsx
<Badge>0</Badge>
<Badge tone="danger">Критичний</Badge>
<Badge tone="ok">OK</Badge>
```

- `tone`: `neutral` (типовий), `ok`, `warn`, `danger`, `info` — кожен забарвлює текст + 12% заливку + бордюр разом.
- Завжди JetBrains Mono, 11px, жирний. Тримайте вміст у межах слова чи числа.
