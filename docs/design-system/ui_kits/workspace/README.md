# UI-кіт — Робоче середовище аналізу

Інтерактивне відтворення операційного інтерфейсу DIMS (`templates/index.html` + `static/style.css`).

- **`index.html`** — точка входу. Завантажує `styles.css`, `workspace.css`, зібраний `_ds_bundle.js` та `app.jsx`.
- **`app.jsx`** — React-застосунок. Компонує примітиви дизайн-системи (`Button`, `Tabs`, `Input`, `Select`, `StatCard`, `GradeBadge`, `SourceItem`, `Alert`, `Badge`) з `window.DIMSDesignSystem_175874`.
- **`workspace.css`** — лише хром сторінки (хедер, двоколонкова сітка, секції панелей, дропзона, зона запуску, результати, критичний банер). Примітиви несуть власні інлайн-стилі.

**Взаємодія:** засіяно корпусом із 3 джерел РФ-медіа. Додайте джерело через вкладку **Текст** → воно з'явиться у списку корпусу. **Запустити аналіз** виконує детермінований мок (нижчий θ + більше джерел → вищий грейд) і рендерить смугу статистики, підозрілі пари, R_DIMS і грейд. Перемикач тем (Авто/Світла/Темна) міняє `[data-theme]` наживо. **Відкрити звіт** веде до кіта звіту.

Математика аналізу — фронтенд-мок для демонстрації; справжнє оцінювання живе в `core/analysis.py` у вихідному репозиторії.
