Параметри аналізу

Рекомендовано: 50–200. Короткі тексти (<500 слів) — 50.

Що це таке?

MFW (Most Frequent Words) — N найчастіших лексем у корпусі. На цьому наборі обчислюється ΔBurrows. Burrows (2002), Eder (2012).

Менше ΔBurrows = більша подібність. Рекомендовано: 0.6–1.0.

Як обрати поріг?

Пари з ΔBurrows < θΔ позначаються як підозріло подібні. Короткі тексти — нижче (0.6–0.7).

Char n-грами (3–4) стійкіші до шуму — Kestemont (2014).

Коли використовувати char n-грами?

Короткі тексти, OCR-помилки, або порівняння перекладів. Для UA/RU новин — слова, n=100.

Виключає слова, що зустрічаються менш ніж у N джерелах.

Навіщо?

Прибирає імена та теми-одноразки. Рекомендовано: 2 для 3–10 джерел. Eder (2013).

MDS точніше відображає Δ-відстані.

Який метод обрати?

PCA — швидкий. MDS — точний для Δ. t-SNE — підкреслює кластери, може спотворювати.
