Сегментований перемикач-пігулка — селектор режиму вводу в лівій панелі (Файли / Текст / Новини / Моніторинг). Контрольований; рендерить лише смугу.

```jsx
<Tabs tabs={[{id:'files',label:'Файли'},{id:'text',label:'Текст'}]} active={tab} onChange={setTab} />
```

- Активна пігулка підіймається до `--bg-1` із волосяним бордюром + `--shadow-sm`; неактивні прозорі до ховера.
- Таби рівної ширини (`flex:1`). Панель під ним рендерите самі.
