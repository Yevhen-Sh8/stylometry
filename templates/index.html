Рекомендовано: 50–200. Для коротких текстів (<500 слів) — 50.

Що це таке?

MFW (Most Frequent Words) — N найчастіших лексем у корпусі. На цьому наборі обчислюється ΔBurrows — середнє абсолютне відхилення z-оцінок частот. Burrows (2002), Eder (2012).

Менше ΔBurrows = більша подібність. Рекомендовано: 0.6–1.0.

Як обрати поріг?

Пари з ΔBurrows < θΔ позначаються як підозріло подібні. На коротких або шумних текстах використовуйте нижчі значення (0.6–0.7); на академічних — до 1.0.

Char n-грами (3–4) стійкіші до шуму та перекладу — Kestemont (2014).

Коли використовувати char n-грами?

Коли тексти короткі, мають помилки OCR, або порівнюються переклади. Для стандартної атрибуції українських/російських новин — слова, n=100.

Виключає слова, що зустрічаються менше ніж у заданій кількості джерел.

Навіщо це потрібно?

Culling прибирає імена власні та теми-одноразки, які спотворюють ΔBurrows. Рекомендовано: 2 для 3–10 джерел, 3–5 для більших корпусів. Eder (2013).

MDS найкраще відображає Δ-відстані; t-SNE — для візуалізації кластерів.

Який метод обрати?

PCA — швидкий, лінійний. MDS — зберігає парні відстані, найточніший для Δ. t-SNE — підкреслює локальні кластери, але може спотворювати загальну структуру.

Класифікація виду прояву DIMs за Методикою. На бали не впливає, окрім виду «Табу», якому автоматично призначається грейд SS.

Перелік видів прояву

Маніпуляція, фейк, інформаційний сендвіч, конспірологія, інсайд, чутлива тема, табу. Якщо вид не встановлено — грейд обчислюється виключно за значенням RDIMS.
