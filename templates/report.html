Методологія

Методика: інтегроване DIMS-оцінювання з використанням стилометричного методу ΔBurrows.

RDIMS = w1Icontent + w2Icoord + w3Idynamics + w4Iimpact + w5Isource

де Icontent, Icoord, Idynamics, Iimpact, Isource — часткові індикатори загрози, а w1 ... w5 — вагові коефіцієнти.

GDIMS = F,   RDIMS < θ1
GDIMS = B,   θ1 ≤ RDIMS < θ2
GDIMS = S,   θ2 ≤ RDIMS < θ3
GDIMS = SS,  θ3 ≤ RDIMS < θ4
GDIMS = SSS, RDIMS ≥ θ4

де θ1 ... θ4 — порогові значення для класифікації рівня інтересу.


Логіка методики: змістові, координаційні, динамічні та впливові ознаки поєднуються в єдину інтегральну оцінку. ΔBurrows використовується як частина індикатора координації між джерелами.


Кластеризація: average linkage використовується для групування джерел за попередньо обчисленою матрицею ΔBurrows.


Обмеження: Метод є стилометричним індикатором, а не доказом. Жанрова схожість або спільна тематика можуть підвищувати подібність незалежних текстів. Результати потребують доповнення контент-аналізом та OSINT-верифікацією.
