Текстове поле з міткою та опційною підказкою. Однорядкове або textarea; моно-варіант для URL / вставленого тексту джерела.

```jsx
<Input label="Назва джерела" placeholder="напр. ТАСС, Джерело А" />
<Input label="Текст" textarea mono placeholder="Вставте текст, HTML або URL…" />
```

- `textarea` → багаторядкове з розтягуванням (мін. 120px). `mono` → JetBrains Mono для коду/URL.
- Фокус малює сигнально-синє кільце 3px. Пара з `hint` — для підказки, а не тултіпа.
