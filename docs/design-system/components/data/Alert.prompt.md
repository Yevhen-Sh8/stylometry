Сповіщення з лівою лінією. Два тони для двох речей, про які попереджає робоче середовище: якість даних (info) та ризик (critical).

```jsx
<Alert tone="critical" title="Змішаний мовний корпус">
  Значення Δ відображають мовну різницю, а не стиль.
</Alert>
<Alert tone="info" title="Короткий текст">Менше 500 слів — надійність знижена.</Alert>
```

- Ліва лінія 3px + 7% тонування в кольорі тону. `info` = бірюза, `critical` = червоний.
- Для банера ескалації, закріпленого вгорі результатів, уживайте патерн `critical-banner` робочого середовища.
