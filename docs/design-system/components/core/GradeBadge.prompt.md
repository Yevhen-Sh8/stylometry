Фірмовий елемент DIMS — інтегральний грейд загрози (F → B → S → SS → SSS, за Наказом МО України № 46). Уживайте всюди, де джерело чи аналіз несе грейд ризику.

```jsx
<GradeBadge grade="SSS" size="lg" meta="0.91" flag />
<GradeBadge grade="S" meta="0.62" />
<GradeBadge grade="F" size="sm" />
```

- Колір фіксований грейдом, не темою: F сірий · B блакитний · S бурштин · SS червоний · SSS темно-карміновий (із внутрішнім кільцем).
- `meta` показує числове R_DIMS поруч із кодом; `flag` додає ⚑ для ескалації SS/SSS.
- Завжди моно, трекінг, табличні. Три розміри: `sm` — інлайн у таблиці, `md` — типовий, `lg` — KPI/заголовок.
