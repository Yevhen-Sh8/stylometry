Грейд DIMS
Виявлено підвищений DIMS-ризик. Інтегральна оцінка 0.7400 → грейд SS. Домінуючі індикатори — змістовий та координаційний аналіз. Дві пари джерел демонструють аномально низьку стилістичну відстань. Рекомендується пріоритетне експертне опрацювання та перевірка скоординованих джерел.

Рекомендації

  1. Провести детальний контент-аналіз пари RT ↔ ТАСС (критичний рівень) для підтвердження координації.
  2. Порівняти часові мітки публікацій підозрілих джерел — синхронність підсилює аргумент координації.
  3. Встановити власників ресурсів через OSINT (WHOIS, MediaOwnership).
  4. Повторити аналіз з різними MFW (50 / 100 / 200) для перевірки стабільності результатів.
  5. Результати є індикативними — інтерпретувати в комплексі з іншими методами.

Методологія

Базова модель: інтегроване DIMS-оцінювання з ΔBurrows як індикатором координації.

RDIMS = w1Icontent + w2Icoord + w3Idynamics + w4Iimpact + w5Isource

Обмеження. Стилометрія є індикатором, а не доказом. Жанрова схожість або спільна тематика можуть підвищувати подібність незалежних текстів. Результати потребують доповнення контент-аналізом та OSINT.
